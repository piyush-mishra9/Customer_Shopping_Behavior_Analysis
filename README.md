# 🛒 Customer Shopping Behavior Analysis

An end-to-end **Data Analytics** project that analyzes customer shopping behavior using **Python, PostgreSQL, SQL, and Power BI**. The project demonstrates the complete analytics workflow—from data loading and preprocessing to SQL-based analysis and interactive dashboard creation.

---

## 📌 Project Overview

The objective of this project is to analyze customer shopping behavior and generate meaningful business insights. The dataset was processed using Python, stored in PostgreSQL, queried with SQL, and finally visualized in Power BI.

This project simulates a real-world analytics workflow used by data analysts.

---

## 🚀 Project Workflow

1. Data Collection
2. Data Cleaning using Python
3. Load Data into PostgreSQL
4. SQL Data Analysis
5. Data Visualization using Power BI
6. Business Insights
